# debt-collective

## Setup

Create `config.js` and `knexfile.js`, edit them as needed.

``sh
cp config/config.sample.js config/config.js
cp knexfile.sample.js knexfile.js
``

Build the application assets before starting.

``sh
npm run build
``

Run the server

``sh
npm start
``
